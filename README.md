# «Гамлет – что посмотреть»
## Телеграмм чат-бот для подбора и поска фильмов
Проект направлен на лёгкий и удобный подбор фильма, просто осуществив ввод преподчтений, а готовая модель искуственного интелекта подберёт максимально подходящий фильм. Также в чат-бот помогает с поиском фильма на сайте «[Кинопоиск](https://www.kinopoisk.ru/)», осуществив ввод названия фильма. Если чат-боту будет введенно сообщение отрицательной тональности, чат-бот оповестит пользователя. Определение тональности происходит благодаря дообученому трансформеру.

## Установка (Windows)
У вас должны быть установлены [зависимости проекта](https://github.com/WannaBeGuineaPig/What_to_watch_on_the_Hamlet#зависимости)

1. Клонирование репозитория 

```git clone https://github.com/WannaBeGuineaPig/What_to_watch_on_the_Hamlet.git```

2. Переход в директорию What_to_watch_on_the_Hamlet

```cd What_to_watch_on_the_Hamlet```

3. Создание виртуального окружения

```python3 -m venv venv```

4. Активация виртуального окружения

```source venv/bin/activate```

5. Установка зависимостей

```pip3 install -r requirements.txt```

## Что доступно в «Гамлет – что посмотреть»?
Данный проект разбит на ряд небольщих библиотек, которые можно использовать друг от друга отдельно.
Библиотеки представленные в данном проекте:
* Телеграмм чат-бот;
* Готовая функция поиска фильмов с запросами к апи «[Кинопоиск](https://www.kinopoisk.ru/)»;
* Готовая функция подбора фильмов, осуществляемый через готовую модель искуственного интелекта;
* Готовая функция определение тональности комментария(для русского языка).
